LeaguePost v4.1.37 CustomTkinter Sidebar Edition

変更点
- CustomTkinterによるChatGPT風サイドバーUIを追加
  - 左上の「三」でサイドバーを開閉
  - 日程、試合速報、順位表、個人賞、設定を画面分離
  - 原稿作成後は別Windowでプレビュー表示
  - 設定画面で文字サイズを変更可能
  - 手入力した大会IDは自動生成で上書きしない
  - 順位表の記録員LINE本文を個人賞へ引き継ぎ可能
- ホーム上部をタイル形式メニューに変更
  - 試合速報
  - 順位＆星取表
  - 個人賞
  - 日程
- 日程ページ作成機能を追加
  - Excelファイル（.xlsx/.xlsm）を選択して、WordPress貼り付け用HTMLを生成
  - 1部/2部の見やすいHTMLテーブルを自動作成
  - 日程表ダウンロードリンクを本文末尾に追加可能
  - 1部/2部それぞれの試合日を記憶し、試合速報作成時の試合日一覧に自動反映
- 2部は「順位＆星取表」のみ生成可能
- 既存のWordPress貼付方式、Chrome拡張機能用データコピー、HTMLブロックコピー方式は維持
- v3.1.2 のタイトル重複解消と、個人賞の続きを読む位置は維持

使い方
1. 上部メニューから作りたいページを選択
2. 年度、季節、区分、大会ID、cupId を入力
3. 試合速報は試合日一覧を入力
4. 順位＆星取表・個人賞は公式記録員LINE本文を貼り付け
5. 日程は「Excelファイルを選択」から日程表ファイルを選択
6. 「原稿生成」を押して、従来どおりWordPressへコピーまたは自動入力

補足
- v4.1の新UIには customtkinter が必要です。未インストールの場合は従来UIで起動します。
- 旧設定の「リーグ戦速報」は v4.0 では「試合速報」として読み替えます。
- 日程ページはExcelファイルから生成します。
