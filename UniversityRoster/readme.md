# 大学名簿統合システム Version 1.0 Release
## 概要

各大学から提出された Excel 名簿を読み込み、統一形式の大学別シートを作成する Windows アプリです。

## 使い方

1. 入力フォルダに各大学の `.xlsx` ファイルを入れる
2. 同じフォルダに `all_import_template.xlsx` を入れる
3. アプリを起動する
4. 入力フォルダを選択する
5. 出力ファイルを指定する
6. 「名簿統合開始」を押す

## 注意事項

- 出力ファイルを Excel で開いたまま実行しないでください
- `all_import_template.xlsx` はテンプレートとして使用されます
- `~$` で始まる一時ファイルは処理対象外です

## 主な仕様

- 大学名はファイル名から取得
- 各大学ファイルの先頭シートを読み込み
- 既存の大学シートは上書き可能
- 背番号は半角数字
- 学年は全角数字＋年
- 氏名は推測しない
- 略名はスペースがある場合のみ姓を使用
- 投打を分離
- 「〃」を直前値で補完
- 選手以外の守備位置は空欄
